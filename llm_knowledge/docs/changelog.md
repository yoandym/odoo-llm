# Changelog

All notable changes to the LLM Knowledge module will be documented in this file.

## [17.0.1.0.0] - 2024-05-01

### Added
- Initial release of the LLM Knowledge module
- Core collection, resource, and chunk models
- Support for multiple embedding models
- Semantic and hybrid search capabilities
- Integration with LLM Assistants
- Document upload and processing
- Basic UI for knowledge management

### Technical Features
- Vector storage in PostgreSQL
- Support for HNSW indexing
- Chunking strategies for different content types
- Embedding caching for improved performance
