OpenAI Provider Module
======================

Overview
--------

The OpenAI Provider module enables integration with OpenAI's language models in Odoo.

Features
--------

* Support for GPT models (GPT-3.5, GPT-4, etc.)
* Chat completions
* Text completions
* Image generation with DALL-E
* Function calling capabilities
* Embeddings for vector search
* Assistants API integration
